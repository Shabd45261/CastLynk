How to Use

Host Mode

  • Launch CastLynk and select Host.
  • Grant Media Projection permission.
  • Enable Accessibility Service for remote control.
  • Stay on the same Wi-Fi network.

Client Mode

  • Launch CastLynk and select Client.
  • Tap Find & Connect to scan.
  • Select the Host from the list.
  • Start controlling the remote device!
